# 🧪 Software Testing – Basics Notes



> **A beginner-friendly, structured reference guide to the fundamentals of Software Testing.**  
> Perfect for students, career switchers, and anyone preparing for QA/testing roles or the ISTQB exam.

---

## 👋 About This Repository

This repository contains well-organised notes on **Software Testing basics**, written in Markdown for easy reading on GitHub.  
Whether you're just getting started in QA or brushing up on core concepts, these notes are designed to be clear, concise, and practical.

### What You'll Find Here

- ✅ Core testing concepts explained simply
- ✅ Types and levels of testing
- ✅ Test case and bug report templates (ready to use)
- ✅ Testing techniques with examples
- ✅ Overview of popular testing tools
- ✅ Agile, TDD & BDD explained
- ✅ A handy cheat sheet for quick revision
